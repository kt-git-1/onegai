# おねがいチャリン Priority 1 ビジュアル基盤

## 1. 文書の位置づけ

Priority 1ワイヤーを実装可能なUI仕様へ進めるための作業文書。

- `確定済み` はユーザー確認済み、または既存仕様で確定している内容
- `要確認` は候補であり、ユーザー確認前に確定扱いしない
- 画面の情報設計と遷移は `futari_charin_priority1_wireframes.md` を正とする

## 4. UIルール

### 4.1 カラートークン

以下をライトテーマの確定値とする。

| Token | Value | 用途 |
|---|---:|---|
| `background` | `#FFF7E8` | 画面背景 |
| `surface` | `#FFFFFF` | カード、入力欄、モーダル |
| `primary` | `#F6B73C` | 主要CTA、選択中セグメント、進捗 |
| `primaryPressed` | `#DEA12E` | 主要CTA押下中 |
| `primarySoft` | `#FFF0C9` | 選択背景、補助面 |
| `accent` | `#FF9A5C` | コインや演出の補助色 |
| `heart` | `#F86F8F` | ちゃりん、スタンプ、重要な感情表現 |
| `textPrimary` | `#4A3426` | 見出し、本文、ボタン文字 |
| `textSecondary` | `#8A7467` | 補足、ラベル、日時 |
| `textDisabled` | `#B9A99F` | disabled文字、非活性アイコン |
| `border` | `#E8DDCE` | 入力欄、小カード、区切り |
| `success` | `#3FA66B` | 成功、プラスのコイン差分 |
| `error` | `#D95763` | 入力・通信エラー、マイナスのコイン差分 |
| `overlay` | `#4A342699` | モーダル背景 |

- 黄色いCTA上の文字は `textPrimary` を使う。
- 色だけで状態を伝えず、文言またはアイコンを併記する。
- ダークテーマはPriority 1の高精細化対象外とし、Color Assetsで差し替え可能な構造だけ維持する。

### 4.2 タイポグラフィ

iOSシステムフォントを使い、Dynamic Typeに対応する。独自フォントは導入しない。

| Style | Size | Weight | 用途 |
|---|---:|---|---|
| `display` | 32 pt | Bold | ちゃりん演出、残高の主表示 |
| `title1` | 24 pt | Bold | オンボーディング見出し |
| `title2` | 20 pt | Semibold | 画面タイトル、カードの主要数値 |
| `headline` | 17 pt | Semibold | セクション見出し、カード名 |
| `body` | 16 pt | Regular | 本文、入力値 |
| `button` | 16 pt | Semibold | 主要・副次ボタン |
| `callout` | 14 pt | Regular | 補足、カード詳細 |
| `caption` | 12 pt | Regular | 日時、短い注釈 |

- 文字間隔は0とし、負のletter spacingは使わない。
- 1行ボタンの文言は原則1行。収まらない場合はボタン幅を優先して調整する。
- 大きな数値は等幅数字を使い、残高変化で横位置が揺れないようにする。

### 4.3 余白と寸法

- 余白スケール：`4 / 8 / 12 / 16 / 24 / 32 pt`
- 画面左右余白：16 pt
- セクション間：24 pt
- 同一セクション内：12 pt
- カード内余白：16 pt
- 最小タップ領域：44 x 44 pt
- 主要ボタン高：52 pt
- 入力欄高：52 pt
- 下部固定CTAはSafe Areaの上に16 pt確保する

### 4.4 角丸

- カード：8 pt
- 入力欄：8 pt
- ボタン：10 pt
- ダイアログ：12 pt
- 円形アイコンボタンは正円にする

### 4.5 境界と影

- 入力欄と小カードは1 ptの `border` を基本にする。
- ホームの貯金箱カードとモーダルのみ弱い影を使う。
- 影：Y 2 pt、Blur 8 pt、黒 8%
- カードを入れ子にせず、行要素は区切り線または面色で整理する。

### 4.6 プロフィール絵文字

- 初期値は未選択。
- 選択は任意。
- 名前が有効なら、絵文字未選択でもCTAをenabledにする。
- 未選択の表示は汎用プロフィールプレースホルダーを使う。

## 5. 状態仕様

各画面について次を定義する。詳細は `futari_charin_priority1_ui_state_spec.md` を正とする。

- 通常
- 押下中
- disabled
- 処理中
- 成功
- 入力エラー / 通信エラー
- 空状態
- 権限差分
- 残高不足 / 交換可能
- 招待待ち / 期限切れ / 参加済み

## 6. 確定履歴

- カラー：A「現行の温かい配色を整理して使用」
- 書体：A「iOSシステムフォント中心」
- 角丸：A「控えめな角丸」
- カード表現：A「境界線中心で主要カードのみ弱い影」
- プロフィール絵文字：任意
- 招待期限切れ：待ち画面内で再発行
- 10秒取り消し：ホーム遷移後も残り時間を継続
- 目標未設定：`ごほうび券を選ぶ` CTAを表示
